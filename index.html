질문수학의 3가지 핵심 철학

🧠

수학은 이해다

공식 암기가 아닌 개념 이해로 시작합니다. 개념 → 질문 → 이해 → 적용의 순서로 수학을 배웁니다.

💬

수학은 언어다

수학을 말로 설명할 수 있어야 진짜 이해한 것입니다. 학생이 개념을 직접 설명하도록 유도합니다.

❓

질문이 사고를 만든다

올바른 질문이 수학적 사고를 유도합니다. 왜 그런지 생각하고 탐구하는 능력을 키웁니다.

6단계 학습 프로세스

과학적으로 설계된 개념 학습 구조

1
Learn — 개념 학습

교사가 수학 개념을 명확하게 설명합니다.

2
Practice — 워크북

학생이 개념을 직접 워크북에 정리합니다.

3
Explain — 설명

학생들이 서로에게 개념을 설명합니다.

4
Repeat — 반복

개념을 반복적으로 설명하며 정착시킵니다.

5
Answer — 질문 답하기

교사의 질문에 학생이 개념으로 답합니다.

6
Solve — 문제 해결

개념 기반으로 수학 문제를 풀이합니다.
